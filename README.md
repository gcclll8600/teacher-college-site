# Teacher College

> 把困惑变成课例，把课例变成研究。

Teacher College 帮助一线教师从真实教学问题出发，借助 AI、同伴共创与持续反馈，形成可实施的课例与持续成长。

## 当前版本

这是 Teacher College 的第一版公开网站，主要用于：

- 说明 Teacher College 的定位与方法；
- 展示四个相互连接的项目原型；
- 公开当前研发重点与待验证问题；
- 为第一轮课例创造营积累潜在参与者。

当前版本为静态前端，不保存用户资料，也不包含账号、支付或学习记录。报名入口将在首轮产品边界确定后接入。

## 本地运行

```bash
npm install
npm run dev
```

## 部署

推送到 `main` 分支后，GitHub Actions 会构建静态站点并部署到 GitHub Pages。
